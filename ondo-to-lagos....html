Connect to Airtable
This app stores all journey data in your Airtable database. You need a personal API token to connect. This is a one-time setup.
How to get your token:
1. Go to airtable.com/create/tokens
2. Click "Create new token"
3. Name it anything (e.g. "Journey App")
4. Scopes: add data.records:read and data.records:write
5. Base access: select "Untitled Base"
6. Click Create token — copy it
7. Paste it below and click Save
Stored locally in your browser only. Never sent anywhere except Airtable.
